Що таке async/await і structured concurrency?

Коротка відповідь

async/await робить асинхронний код схожим на синхронний, а structured concurrency задає життєвий цикл задач через ієрархію: child tasks прив'язані до parent task.

Варто згадати cancellation propagation, TaskGroup, MainActor для UI і те, що async не означає автоматично background thread.

@MainActor
final class UsersViewModel: ObservableObject {
    @Published private(set) var users: [User] = []

    func load() async throws {
        users = try await service.fetchUsers()
    }
}
