Що таке copy-on-write?

Коротка відповідь

Copy-on-write означає, що копія value type фактично не дублює storage, доки один із власників не спробує змінити значення. Так працюють Array, Dictionary, Set і String.

Це дає зручну семантику value types без постійних дорогих копій великих буферів.

var first = [1, 2, 3]
var second = first
second.append(4) // storage копіюється перед зміною
