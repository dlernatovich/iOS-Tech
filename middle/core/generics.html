Для чого потрібні generics та associated types?

Коротка відповідь

Generics дозволяють писати reusable, type-safe код без втрати інформації про тип. associatedtype використовується в protocols, коли конкретний тип має визначити частину контракту.

Сильна відповідь включає різницю між generic constraints, opaque types some і existential types any.

protocol Repository {
    associatedtype Entity
    func load(id: String) async throws -> Entity
}
