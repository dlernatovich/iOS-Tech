Middle / UI

Як працює data flow у SwiftUI?

Коротка відповідь

Основні інструменти: @State для локального стану, @Binding для двостороннього зв'язку, @StateObject для власного observable object, @ObservedObject для переданого object, @EnvironmentObject для shared state.

Головна ідея: UI перераховується при зміні state, тому side effects треба ізолювати у view models/services.
