Middle / UI

Для чого потрібен UICollectionViewCompositionalLayout?

Коротка відповідь

Compositional Layout дозволяє будувати складні layouts з sections, groups, items, supplementary views і orthogonal scrolling. Він добре підходить для modern home/feed/catalog screens.

Кандидат має пояснити hierarchy item → group → section → layout і trade-off між гнучкістю та складністю конфігурації.
