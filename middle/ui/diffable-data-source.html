Що таке Diffable Data Source?

Коротка відповідь

Diffable Data Source керує даними таблиці або колекції через snapshots. UIKit сам рахує різницю між старим і новим станом та анімує insert/delete/move.

Моделі мають мати стабільну ідентичність через Hashable. Це зменшує кількість багів із ручним performBatchUpdates.

var snapshot = NSDiffableDataSourceSnapshot<Section, Item>()
snapshot.appendSections([.main])
snapshot.appendItems(items)
dataSource.apply(snapshot, animatingDifferences: true)
