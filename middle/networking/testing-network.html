Middle / Networking

Як тестувати networking layer?

Коротка відповідь

Networking layer тестують через protocol abstraction, mock clients, custom URLProtocol, fixtures для JSON і перевірку mapping errors. Unit tests не мають залежати від реальної мережі.

protocol HTTPClient {
    func data(for request: URLRequest) async throws -> (Data, HTTPURLResponse)
}

Integration tests з реальним backend можуть бути окремими і менш частими.
