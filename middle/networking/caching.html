Як працює HTTP caching в iOS?

Коротка відповідь

HTTP caching керується headers: Cache-Control, ETag, Last-Modified, Expires. URLCache може кешувати responses, якщо конфігурація і headers це дозволяють.

Для app-specific offline UX часто потрібен окремий persistent cache або database, бо HTTP cache не замінює доменне сховище.
