Як працює refresh token flow?

Коротка відповідь

Коли access token протермінований, client отримує 401, виконує refresh через refresh token, зберігає нові credentials і повторює original request. Важливо синхронізувати одночасні refresh operations, щоб не запускати refresh для кожного паралельного request.

Refresh token треба зберігати безпечно, зазвичай у Keychain.
