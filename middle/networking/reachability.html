Middle / Networking

Як працювати з offline та reachability?

Коротка відповідь

NWPathMonitor дозволяє спостерігати за станом мережі, але не гарантує, що конкретний API доступний. Reachability варто використовувати для UX hints і queueing, а не як єдине джерело істини перед request.

Надійний код все одно має обробляти помилки кожного network call.
