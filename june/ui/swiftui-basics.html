Що таке View, @State та @Binding у SwiftUI?

Коротка відповідь

View описує UI як функцію від стану. @State зберігає локальний стан view. @Binding передає можливість читати і змінювати стан, який належить іншій view.

struct CounterView: View {
    @State private var count = 0

    var body: some View {
        Button("Count: \(count)") {
            count += 1
        }
    }
}
