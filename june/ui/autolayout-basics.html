Що таке Auto Layout constraints?

Коротка відповідь

Auto Layout описує позицію та розмір view через constraints: відступи, ширину, висоту, alignment, relationship між елементами. Система розв'язує ці рівняння для різних екранів і орієнтацій.

Часті проблеми: ambiguous layout, unsatisfiable constraints, забутий translatesAutoresizingMaskIntoConstraints = false.

button.translatesAutoresizingMaskIntoConstraints = false
NSLayoutConstraint.activate([
    button.centerXAnchor.constraint(equalTo: view.centerXAnchor),
    button.bottomAnchor.constraint(equalTo: view.safeAreaLayoutGuide.bottomAnchor, constant: -24)
])
