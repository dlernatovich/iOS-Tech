Коли обрати UITableView, а коли UICollectionView?

Коротка відповідь

UITableView добре підходить для простих вертикальних списків. UICollectionView гнучкіший: grid, horizontal sections, nested layouts, compositional layout.

У сучасному UIKit для складних екранів частіше обирають UICollectionView, навіть якщо візуально це список.
