Які HTTP basics має знати iOS Developer?

Коротка відповідь

Потрібно розуміти methods (GET, POST, PUT, PATCH, DELETE), headers, body, query parameters, status codes, JSON, TLS і caching headers.

Для мобільної розробки особливо важливі timeout, retries, idempotency, auth headers і коректна обробка нестабільної мережі.
