Як працює Codable?

Коротка відповідь

Codable це typealias для Encodable & Decodable. Він дозволяє перетворювати Swift models у JSON і назад. Для нестандартних назв полів використовують CodingKeys.

struct UserDTO: Decodable {
    let id: Int
    let fullName: String

    enum CodingKeys: String, CodingKey {
        case id
        case fullName = "full_name"
    }
}
