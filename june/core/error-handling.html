Як працює error handling у Swift?

Коротка відповідь

Swift використовує throw, throws, do/catch, try, try? і try!. Помилки часто описують enum, який conform до Error.

На практиці важливо мапити низькорівневі помилки в доменні, щоб UI не знав деталей мережі, бази даних чи JSON.

enum LoginError: Error {
    case invalidCredentials
    case serverUnavailable
}

func login() throws {
    throw LoginError.invalidCredentials
}
