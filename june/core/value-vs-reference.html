Чим value types відрізняються від reference types?

Коротка відповідь

struct, enum і tuple у Swift є value types: при присвоєнні або передачі створюється логічна копія значення. class є reference type: змінні зберігають посилання на один і той самий об'єкт.

На співбесіді важливо згадати мутабельність, ARC для класів, copy-on-write в стандартних колекціях і те, що value types часто простіше тестувати та безпечніше використовувати в конкурентному коді.

struct Profile {
    var name: String
}

var a = Profile(name: "Ira")
var b = a
b.name = "Oleh"

print(a.name) // Ira
print(b.name) // Oleh
