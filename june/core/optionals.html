Що таке Optional і як безпечно дістати значення?

Коротка відповідь

Optional<T> означає, що значення типу T може бути відсутнім. Безпечні способи роботи: if let, guard let, optional chaining, nil-coalescing ??.

Force unwrap ! варто використовувати тільки коли інваріант гарантований і падіння програми буде коректним сигналом помилки розробника.

func title(from response: [String: String]) -> String {
    guard let title = response["title"], !title.isEmpty else {
        return "Untitled"
    }
    return title
}
