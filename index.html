iOS Interview Theory

Питання та короткі відповіді для співбесіди Senior iOS Developer

Матеріали розбиті за рівнями та групами. Кожне питання живе в окремому HTML-файлі, тому його легко редагувати, доповнювати прикладами коду або переносити між секціями.

Як користуватись сайтом для навчання

Починай із Junior-рівня, навіть якщо готуєшся на Senior: базові питання часто відкривають прогалини у фундаменті. Для кожної теми спробуй спочатку відповісти вголос за 1-2 хвилини, потім відкрий сторінку, звір коротку відповідь і пройди навчальні блоки: суть, як пояснювати на співбесіді, типові помилки та практику.

Для Senior-підготовки особливо корисно не просто знати визначення, а вміти говорити про trade-offs: продуктивність, тестованість, підтримку, безпеку, міграції та роботу команди в довгостроковому проєкті.

Level 1

June / Junior

Core

  • Value types vs reference types
  • Optionals та optional binding
  • Protocols і extensions
  • ARC, strong, weak, unowned
  • Error handling у Swift

UI

  • UIViewController lifecycle
  • Auto Layout basics
  • UITableView vs UICollectionView
  • Storyboard, XIB чи UI in code
  • SwiftUI basics: View, State, Binding

Networking

  • HTTP basics для iOS
  • URLSession базово
  • Codable parsing
  • Status codes та помилки API
  • Простий API client

Level 2

Middle

Core

  • Generics та associated types
  • async/await і structured concurrency
  • GCD vs OperationQueue
  • Copy-on-write
  • Dependency Injection

UI

  • Diffable Data Source
  • Compositional Layout
  • SwiftUI data flow
  • Accessibility basics
  • Navigation і Coordinators

Networking

  • Retry, timeout, backoff
  • Auth і refresh token flow
  • HTTP caching
  • Reachability та offline states
  • Тестування networking layer

Level 3

Seniour / Senior

Core

  • Actors, Sendable, data races
  • Memory graph та leak hunting
  • Architecture trade-offs
  • Modularization і build times
  • Performance profiling з Instruments

UI

  • SwiftUI та UIKit interoperability
  • Rendering performance
  • Design system для iOS
  • State restoration
  • UI testing strategy

Networking

  • Networking architecture
  • TLS, certificate pinning, App Transport Security
  • REST vs GraphQL vs gRPC
  • Background transfers
  • Observability networking layer
