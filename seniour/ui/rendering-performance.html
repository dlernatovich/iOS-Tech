Як діагностувати проблеми rendering performance?

Коротка відповідь

Проблеми видно як dropped frames, stutters, slow scrolling. Причини: main-thread work, важкий Auto Layout, sync image decoding, overdraw, offscreen rendering, занадто часті SwiftUI updates.

Інструменти: Core Animation, Time Profiler, os_signpost, Xcode View Debugger. Рішення: prefetching, caching, background decoding, зменшення invalidations, стабільна ідентичність views/items.
