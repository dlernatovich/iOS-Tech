Як поєднувати SwiftUI та UIKit у production app?

Коротка відповідь

UIHostingController вбудовує SwiftUI в UIKit, а UIViewRepresentable/UIViewControllerRepresentable вбудовують UIKit у SwiftUI. Важливо контролювати navigation ownership, lifecycle, sizing, state synchronization і dependency injection.

let controller = UIHostingController(
    rootView: ProfileView(viewModel: viewModel)
)
navigationController.pushViewController(controller, animated: true)
