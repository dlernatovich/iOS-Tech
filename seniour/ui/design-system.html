Seniour / UI

Що входить у сильну iOS design system?

Коротка відповідь

Design system містить tokens для color/typography/spacing, reusable components, accessibility rules, theming, localization behavior, snapshot tests і документацію для дизайнерів та розробників.

Senior має думати про API компонентів, backward compatibility, adoption plan, Figma parity, dark mode, Dynamic Type і механізм міграцій.
