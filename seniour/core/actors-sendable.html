Seniour / Core

Actors, Sendable і data races: що важливо знати Senior iOS Developer?

Коротка відповідь

actor ізолює mutable state і дозволяє безпечний доступ через async boundary. Sendable позначає типи, які безпечно передавати між concurrency domains. Це допомагає компілятору ловити data races.

Senior має пояснити actor reentrancy, MainActor, unchecked Sendable, ізоляцію mutable state і межі, де потрібні locks або serial executors.

actor TokenStore {
    private var token: String?

    func update(_ value: String) {
        token = value
    }

    func current() -> String? {
        token
    }
}
