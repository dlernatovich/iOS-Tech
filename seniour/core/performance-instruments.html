Seniour / Core

Як профілювати performance через Instruments?

Коротка відповідь

Спершу формулюють гіпотезу і вимірюють на реальному девайсі. Time Profiler шукає CPU hotspots, Allocations і Leaks - пам'ять, Network - requests, Energy Log - battery impact, Core Animation - dropped frames.

Senior має говорити про baseline, reproducible scenario, release build, symbols, main-thread blocking і перевірку після оптимізації.
